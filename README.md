#  Inventory Management System

A web-based Inventory Management System developed to help organizations manage and track office assets efficiently.

---

##  Project Description

This system allows users to manage inventory records such as office equipment, track availability, and maintain proper asset records within an organization.

The project was developed as part of practical training and personal development in web application development.

---

##  Features

- Add new inventory items
- Update and delete items
- View all stored assets
- Department-based asset management
- User authentication system 
- Responsive web interface

---

##  Technologies Used

- Java (Spring Boot)
- PostgreSQL
- HTML
- CSS
- JavaScript
- Bootstrap
- jQuery

---

##  Project Structure

- `src/` - Main source code
- `controller/` - Application controllers
- `service/` - Business logic
- `repository/` - Database layer
- `templates/` - HTML pages
